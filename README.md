# Overview 

Btd6Helper is a program that can help you calculate the damage and income 
of towers in BTD6. 

# How to use

Read the user guide at [docs/UserGuide.md](docs/UserGuide.md)

# Currently supported towers

- Dart Monkey (233)
- Boomerang Monkey (433)
- Bomb Shooter (331)
- Tack Shooter (335)
- Sniper Monkey (303)
- Monkey Sub (000)
- Monkey Buccaneer (000)
- Monkey Ace (000)
